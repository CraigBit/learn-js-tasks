<em>2.11 TASK 9</em>

<p>Напишите код, который будет спрашивать логин с помощью prompt.</p>

<p>
	Если посетитель вводит «Админ», то prompt запрашивает пароль, если ничего
	не введено или нажата клавиша Esc – показать «Отменено», в противном
	случае отобразить «Я вас не знаю».
</p>

<p>Пароль проверять так:</p>

<ul>
	<li>Если введён пароль «Я главный», то выводить «Здравствуйте!»,</li>
	<li>Иначе – «Неверный пароль»,</li>
	<li>При отмене – «Отменено».</li>
</ul>

<p>
	Для решения используйте вложенные блоки if. Обращайте внимание на стиль и
	читаемость кода.
</p>

<p>
	Подсказка: передача пустого ввода в приглашение prompt возвращает пустую
	строку ''. Нажатие клавиши Esc во время запроса возвращает null.
</p>

  обработку нечисловых строк в этой задаче необязательно.
</p>
